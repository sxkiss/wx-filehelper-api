# plugins

插件目录（按文件夹组织），用于扩展命令、消息处理器和 HTTP 路由；部分插件也包含静态资源（WebUI）。

## Files

| File | Role | Function |
|------|------|----------|
| `builtin/` | Plugin | 内置命令集 |
| `example/` | Plugin | 示例插件（演示命令/路由扩展） |
| `framework_api/` | Plugin | 框架管理接口（状态、任务、插件管理等） |
| `webui/` | Plugin | Web 管理界面（扫码登录、状态监控） |
